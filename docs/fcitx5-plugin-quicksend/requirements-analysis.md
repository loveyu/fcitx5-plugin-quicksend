# 快捷发送插件 - 需求分析

## 1. 功能概述

在输入法工具栏添加一个快捷发送按钮，用于快速发送预设的快捷键/文本组合（如 `Ctrl+Shift+Del`、`Shift+Tab` 等）。支持多种发送模式，并提供完整的管理界面。

## 2. 核心功能

### 2.1 发送触发
- 在输入法工具栏上显示快捷发送图标按钮
- 点击按钮弹出快捷发送列表，选择条目后执行发送

### 2.2 发送内容类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 快捷键组合 | 多键同时按下 | `Ctrl+Shift+Del` |
| 单键 | 单个按键 | `Tab`, `Escape` |
| 文本句子 | 发送一段文字 | `你好世界` |
| 混合 | 以上任意组合 | 任意 |

### 2.3 发送模式
- **一起发送（组合键模式）**：类似真实键盘同时按下，适用于 `Ctrl+Shift+Del` 这类快捷键
- **单个发送（序列模式）**：按顺序逐个发送字符/按键，适用于文本句子或需要依次触发的场景

每种发送条目可配置其发送模式。

### 2.4 设置页面
插件提供独立的设置页面，支持以下管理功能：

#### 条目管理
- 新增快捷发送条目
- 编辑已有条目
- 删除条目

#### 编辑器功能
- 输入发送内容
- **特殊字符下拉**：编辑时提供下拉列表，可选择键盘特殊键（如 `Ctrl`, `Shift`, `Alt`, `Tab`, `Escape`, `Enter`, `Backspace`, `Del`, `Insert`, `Home`, `End`, `PageUp`, `PageDown`, 方向键等）
- 配置发送模式（一起发送 / 单个发送）

#### 排序
- 默认按使用次数倒序排列（使用最多的排最前）
- 使用次数相同时，可再按添加时间或自定义排序

#### 计数器
- 每使用一次发送该条目，计数器 +1
- 编辑时**允许手动调整**使用次数（应对导入迁移等场景）

### 2.5 容量限制
- 最大快捷发送条目数：**500 条**
- 一次性全量加载，**不涉及分页**相关逻辑

## 3. 数据模型

### 3.1 核心实体

```kotlin
data class QuickSendEntry(
    val id: Long = 0,
    val label: String = "",              // 显示名称（选填）
    val segments: List<ContentSegment>,  // 内容段列表
    val sendMode: Int,                   // 0=COMBINATION, 1=SEQUENCE
    val useCount: Int = 0,
    val createdAt: Long = 0,
    val updatedAt: Long = 0
)
```

### 3.2 内容段

```kotlin
data class ContentSegment(
    val type: Int,      // 0=普通文本, 1=特殊键, 2/3=预留
    val content: String  // type=0 文本原文; type=1 键名（如 "CTRL", "SHIFT"）
)
```

### 3.3 显示策略

| 条件 | 列表/窗口显示 |
|------|-------------|
| `label` 非空 | 显示 `label` |
| `label` 为空 | 遍历 segments: type=0 直接显示文本; type=1 带边框显示 `CTRL` |

### 3.4 段合并发送规则

**COMBINATION（一起发送）**:
- 连续的 modifier type=1 段合为修饰键列表
- 第一个非 modifier type=1 作主键（同时按下 modifiers + 主键）
- 剩余 type=0 合并为文本追加
- 剩余 type=1 回退为单个按键

**SEQUENCE（单个发送）**:
- 每个段独立发送
- type=0: 逐字符 commitText
- type=1: 单个 key down/up

## 4. UI 交互流程

```
工具栏按钮 → 弹出列表 → 选择条目 → 发送
                          → 计数器 +1
                          → 列表排序更新
```

```
设置页面 → 条目列表（useCount 倒序） → 点击编辑
                                     → 段编辑器
                                       → 内容段 FlowLayout
                                       → type=1 段带边框显示
                                       → 特殊字符下拉选择插入 type=1 段
                                       → 发送模式选择
                                       → 使用次数调整
```

## 5. 边界与约束

| 约束项 | 说明 |
|--------|------|
| 最大条目数 | 500 条 |
| 加载方式 | 全量加载，无分页 |
| 排序规则 | 按使用次数倒序 |
| 发送模式 | 每条条目必须明确指定一种模式 |
| 特殊字符 | 编辑时提供分组下拉选择，选中后插入 type=1 段 |
| type=1 显示 | 带边框 UI 区分于普通文本 |
| 段类型 2/3 | 预留，暂不处理 |
